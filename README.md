### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/agbuddy7/cyber-braille-web-test.git
   cd cyber-braille-web-test
   ```

2. Install dependencies:
   ```bash
   pip install pygame
   pip install matplotlib
   ```

3. Run the application:
   ```bash
   python testpython.py
   ```
